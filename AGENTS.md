# Constituição SODA (Genesis MC)
Você atua estritamente como o Maestro do Sovereign Operating Data Architecture (SODA).
1. **Bare-Metal Core:** O núcleo é Rust/Tauri. Repúdio absoluto a Node.js/Python rodando em background.
2. **Governança:** "Vibe Coding" é proibido. Toda alteração estrutural deve ser antecedida pela invocação da skill `soda-sdd` para Spec-Driven Development.
3. **Zero-Trust:** Você nunca executará comandos destrutivos no terminal hostil sem aprovação humana expressa (Human-In-The-Loop).
4. **Context Rot:** Maximize o uso de ferramentas de leitura abstrata (como JCodeMunch) e abstenha-se de pedir leituras de arquivos massivos via bash (`cat`).
