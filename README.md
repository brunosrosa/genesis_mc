# 🌌 Genesis - Mission Control

**Genesis** é um Sistema Operacional Agêntico Soberano e Orquestrador de IA, projetado para atuar como um *Life OS* local, contínuo e altamente visual. Construído para mentes de alto desempenho, ele substitui a fadiga cognitiva de logs de terminal por interfaces espaciais fluidas.

## 🚀 Arquitetura Tecnológica

Este projeto rejeita a dependência exclusiva de nuvem e as arquiteturas baseadas em Node.js/Electron em favor de um ecossistema nativo de altíssima performance:

- **Frontend (A Lente):** React 18, Vite, TypeScript.
- **Backend (O Motor):** Rust (Tauri v2), operando o núcleo de orquestração *ZeroClaw*.
- **Comunicação:** Tauri IPC (Inter-Process Communication) orientada a eventos.
- **UI & UX:** Tailwind CSS, Shadcn UI, Framer Motion, React Flow e Pragmatic Drag & Drop.
- **Inteligência:** Integração agnóstica de LLMs (Locais via Ollama/LM Studio e Cloud via Gemini/Claude) orquestrada por MCP (Model Context Protocol).

## 🧠 Filosofia

O Genesis baseia-se no conceito de **"Human-in-the-Loop"** e **Acessibilidade Cognitiva**. O agente orquestrador prepara fluxos de trabalho, escreve código, analisa dados e navega na web de forma autônoma (usando o padrão ReAct e Ralph Loops), mas nunca cruza os portões de segurança físicos ou financeiros sem a aprovação explícita do Diretor (Você) através do painel de controle de missão.
