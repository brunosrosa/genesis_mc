# 🌌 Genesis - Mission Control

**Genesis** é um Sistema Operacional Agêntico Soberano e Orquestrador de IA, projetado para atuar como um *Life OS* local, contínuo e altamente visual. Construído para mentes de alto desempenho, ele substitui a fadiga cognitiva de logs de terminal por interfaces espaciais fluidas, garantindo total privacidade dos dados.

## 🚀 Arquitetura Tecnológica

Este projeto rejeita a dependência exclusiva de nuvem e as arquiteturas web tradicionais em favor de um ecossistema nativo de altíssima performance para Windows:

- **Frontend (A Lente):** React 18, Vite, TypeScript.
- **Backend (O Motor):** Rust (Tauri v2), operando o núcleo de orquestração *ZeroClaw*.
- **Memória Bitemporal:** Ecossistema `Beads` rodando sobre banco de dados **Dolt** (MySQL 3306).
- **UI & UX:** Tailwind CSS v4, Shadcn UI, Framer Motion, React Flow e Pragmatic Drag & Drop.

## 🧠 Filosofia e Operação

O Genesis baseia-se no conceito de **"Human-in-the-Loop"**. O agente orquestrador prepara fluxos de trabalho, analisa dados e gera código de forma autônoma (usando o motor ARC-Ralph e execução Stateless), mas NUNCA cruza os portões de segurança físicos (file system) ou financeiros sem a aprovação explícita do Diretor através do painel de controle.
