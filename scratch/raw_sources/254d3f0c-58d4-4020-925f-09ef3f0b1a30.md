Context Protocol (MCP) server that provides web search capabilities through DuckDuckGo, with additional features for content fetching and parsing.\n\nQuick Start\n\nuvx duckduckgo-mcp-server\n\n\nFeatures\n\nWeb Search\n\n: Search DuckDuckGo with advanced rate limiting and result formatting\n\nContent Fetching\n\n: Retrieve and parse webpage content with intelligent text extraction\n\nRate Limiting\n\n: Built-in protection against rate limits for both search and content fetching\n\nError Handling\n\n: Comprehensive error handling and logging\n\nLLM-Friendly Output\n\n: Results formatted specifically for large language model consumption\n\nInstallation\n\nInstall from PyPI using \n\nuv\n\n :\n\nuv pip install duckduckgo-mcp-server\n\n\nUsage\n\nRunning with Claude Desktop\n\nDownload \n\nClaude Desktop\n\nhttps://claude.ai/download\n\nCreate or edit your Claude Desktop configuration:\n\nOn macOS: \n\n~/Library/Application Support/Claude/claude_desktop_config.json\n\nOn Windows: \n\n%APPDATA%\\Claude\\claude_desktop_config.json\n\nAdd the following configuration:\n\nBasic Configuration (No SafeSearch, No Default Region):\n\n{\n    \"mcpServers\": {\n        \"ddg-search\": {\n            \"command\": \"uvx\",\n            \"args\": [\"duckduckgo-mcp-server\"]\n        }\n    }\n}\n\n\nWith SafeSearch and Region Configuration:\n\n{\n    \"mcpServers\": {\n        \"ddg-search\": {\n            \"command\": \"uvx\",\n            \"args\": [\"duckduckgo-mcp-server\"],\n            \"env\": {\n                \"DDG_SAFE_SEARCH\": \"STRICT\",\n                \"DDG_REGION\": \"cn-zh\"\n            }\n        }\n    }\n}\n\n\nConfiguration Options:\n\nDDG_SAFE_SEARCH\n\n : SafeSearch filtering level (optional)\n\nSTRICT\n\n : Maximum content filtering (kp=1)\n\nMODERATE\n\n : Balanced filtering (kp=-1, default if not specified)\n\nOFF\n\n : No content filtering (kp=-2)\n\nDDG_REGION\n\n : Default duckduckgo-mcp-server\n\nRunning with SSE or Streamable HTTP\n\nThe server supports alternative transports for use with other MCP clients:\n\n# SSE transport\nuvx duckduckgo-mcp-server --transport Search Tool\n\nasync def search(query: str, max_results: int = 10, region: str = \"\") -> str\n\n\nPerforms a web search on DuckDuckGo and returns formatted results.\n\nParameters:\n\nquery\n\n : Search query string\n\nmax_results\n\n : Maximum number of results to return (default: 10)\n\nregion\n\n : (Optional) Region/language code to override the default.
